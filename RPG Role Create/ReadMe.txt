	☆☆☆☆★使い方★☆☆☆☆
① 本製品(XPM.exe)を起動します。
② メニューから [オプション]->[設定] を選択します。
③ 各種設定を変更します。
④ 画面左の一覧からカテゴリーを選択し、右クリックメニューから [プロジェクト検索] を選択します。
⑤ 表示されたダイアログに各種値を設定し、[検索開始] ボタンを押します。
⑥ これでリストにプロジェクトが登録されます。
　 あとはリスト項目を右クリックすることで、各種操作を行うことができます。

　※プロジェクトを直接リスト上にドラッグしても追加できます。


	☆☆☆☆★バックアップについて★☆☆☆☆
 本製品の初回起動時、自動的にバックアップフォルダが
　[ <起動パス>\BackUp ]
に設定されます。
 このままではいかん、という場合は [オプション]->[設定] から変更してください。

 バックアップファイルは
　[<バックアップフォルダ>\<ゲームタイトル>]
というフォルダに
　[yyyy-MM-dd_hh-mm-ss.<拡張子>]
というファイル名で作成されます。
 例えば『Project1』を「2004年12月11日 23時15分57秒」にLHAでバックアップした場合、
　[<バックアップフォルダ>\Project1\2004-12-11_23-15-57.lzh]
となります。

 バックアップに使用できる形式は
┏━━━━━━━━━━━━━━━━━━┓
┃【LHA(LZH)書庫】【LHA自己解凍書庫】 ┃
┃【ZIP書庫】　　　　　　　　　　　　 ┃
┃【CAB書庫】【CAB自己解凍書庫】　　　┃
┗━━━━━━━━━━━━━━━━━━┛
の５つです。
(【ZIP自己解凍】は面倒臭いので現段階では未対応)
 設定ウィンドウで形式を選択した際に「-Not Found-」と出るものは、圧縮に必要なDLLが
導入されていないため、使用することはできません。
 必要なDLLは『統合アーカイバ・プロジェクト』からDLすることができます。
　※DLLが一つも入っていない場合、バックアップ機能は使用できません。

 それぞれの形式には特徴がありますので、好みに合わせて使用してください。


	☆☆☆☆★圧縮形式簡易比較★☆☆☆☆
　　▼圧縮率(高いほどサイズが小さくなる)▼
１：CAB(LZX)
２：CAB(MSZIP)
３：ZIP
４：LHA
 ↑はあくまで一般的な順位。状況によって多少変動。
 (例えば、RGSSサンプルだとZIPよりLHAの方が高圧縮)
 CABのLZXはレベル指定が可能なので、かなりの高圧縮率が期待できる。
 ただし圧縮時間は長くなるので、その辺は好みで。
　□ 参考までに『Zeit von Schicksal』(2004/12/11現在)のファイルサイズ □
　　無圧縮：11.0MB
　　　 LHA：7.90MB
　　ZIP(1)：8.05MB
　　ZIP(9)：7.95MB
CAB(MSZIP)：7.73MB
CAB(LZX15)：7.65MB
CAB(LZX21)：7.52MB
　ZIPレベル0は無圧縮(まとめるだけ)なので、あえてレベル1で実験しています。
　意外にも【CAB(MSZIP)】が最速。当然ながら【CAB(LZX21)】が最も遅い。
　そしてZIPに至っては、レベル6～9で変わらないという結果が…。
　ZIPはゲームに向いていない…とか？

　　▼汎用性▼
１：LHA … 日本では最も標準的。対応ソフトも多い。
２：ZIP … 米国での標準形式？Me以降の「圧縮フォルダ」もコレ。
３：CAB … 主にMicrosoft社が配布物に使用する形式。
　　　　　 高圧縮率だが、書庫へのファイル追加は不可能。
　　　　　(と言っても本製品では問題無し)

 ↑をご覧頂ければ分かる通り、完全に好みの問題です。
 普通はLHA、少しでも容量を節約したい場合はCAB…？
